# messenger-app

## Usage
* Clone this repository
* Install the depencies by the following commands:
  ```
  npm run build
  ```
* Run the app by the following script:
  ```
  npm start
  ```
  it will start the server and a client simultaneously
* The default client is on `localhost:3000`
